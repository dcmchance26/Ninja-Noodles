Ninja Noodles

Fight. Upgrade. Become Legendary.

Ninja Noodles is a mobile-first 2D fighting RPG where rival noodle clans battle for honor, rare loot, and the title of Grand Noodle Master.

Players can choose from multiple noodle classes, unlock powerful equipment, defeat increasingly difficult opponents, and climb the ranks through skill and progression.

Planned Features

* Mobile touch controls
* Best-of-three fighting matches
* Multiple noodle classes
* Equipment and cosmetics
* Rare, Epic, and Legendary loot
* Boss battles
* Character progression
* Browser-based gameplay
* Online multiplayer (planned)

Starter Classes

* Ramen (Balanced)
* Spaghetti (Fast)
* Fettuccine (Strong)

Development Status

Pre-Alpha
